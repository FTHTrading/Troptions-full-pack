# TTN TV (frontend slice)

Next.js routes and registries extracted from `troptions` Exchange OS:

- `src/app/ttn/` — channels, studio, proof, sponsors
- `src/components/ttn/`
- `src/content/ttn/`, `src/data/ttn/`

Run the full app from `frontends/exchange-os` (`npm run dev`). This folder is the TTN-specific surface for independent packaging.

API backend: `backend/ttn-launcher` on port **8092**.
