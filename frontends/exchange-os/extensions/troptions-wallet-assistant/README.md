# Troptions Wallet Assistant

Chrome extension helper for the local Troptions wallet simulation environment.

## Purpose

- Open the restored wallet routes quickly while working on the local build.
- Keep the default base URL on `http://localhost:8889`.
- Expose only simulation-safe routes.

## Included shortcuts

- `/troptions`
- `/join-troptions`
- `/join-troptions/create-wallet`
- `/portal/troptions/wallet/dashboard`
- `/portal/troptions/wallet/send`
- `/portal/troptions/wallet/convert`
- `/portal/troptions/wallet/x402`
- `/admin/troptions/wallets`
- `/admin/troptions/navigation-audit`
- `/admin/troptions/openclaw/dashboard`
- `/admin/troptions/openclaw/jefe`
- `/admin/troptions/openclaw/site-ops`
- `/admin/troptions/openclaw/x402`
- `/admin/troptions/xrpl-platform/mainnet-readiness`

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer Mode.
3. Click `Load unpacked`.
4. Select this folder.

All wallet actions remain simulation-only.
