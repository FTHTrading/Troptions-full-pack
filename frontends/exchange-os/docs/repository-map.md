# Troptions Repository Map

- `src/app`: Next.js App Router pages and API routes for institutional surfaces and control-plane endpoints.
- `src/components`: Shared UI components for institutional dashboards, portal views, and disclosure surfaces.
- `src/content/troptions`: Registry-driven content models for claims, approvals, rails, XRPL, trading, and governance states.
- `src/lib/troptions`: Domain engines and guard logic for compliance gates, workflows, observability, and safe simulations.
- `src/__tests__/troptions`: Jest tests for policy gates, route guards, observability, safety constraints, and API behavior.
- `scripts`: Operational automation for migrations, backups, cutover drills, recovery timing, and smoke checks.
- `docs/runbooks`: Operator runbooks for deployment, key rotation, incident response, and database procedures.
- `public/troptions`: Public static brand assets, media, and report imagery.
