# Yield Separation Algorithm

The key routing question is whether the proposed value accrual belongs to:

- payment stablecoin settlement,
- tokenized deposits under FI authority,
- merchant-funded transaction rebates,
- non-cash loyalty,
- reserve income retained by issuer,
- or a prohibited holder-yield structure.

## Core Rule

If value is paid solely because someone holds, uses, or retains a payment stablecoin, the structure is blocked or treated as highly risky.

## Safer Alternatives

- Tokenized deposit lane with regulated FI controls
- Merchant-funded purchase rebate
- Non-cash loyalty tied to commerce
- Software and service fees paid to Troptions
