**Audited:** 2025  
**Status:** Internal compliance review — NOT for public distribution  
**Purpose:** Identify, classify, and prescribe remediation for every claim that requires modification before any public, commercial, or technical deployment of Momentum-branded content.

## Prohibited Claims — Zero Tolerance

These claims **cannot appear** in any public, commercial, or technical output:

1. Fractional ownership of venue/revenue tokens sold to public without securities registration
2. Revenue-sharing or "digital dividend" distributed to token holders without SEC registration
3. Performance-linked investment contracts (athlete token payouts to investors)

## Categories of Required Review (Before Any Feature Activation)

| Category | Authority | Notes |
|----------|-----------|-------|
| Securities (tokens, fractional ownership, investment contracts) | SEC / state securities regulators | Howey test applies to all token structures |
| Money Transmission (payments, payroll, stablecoin) | FinCEN + state money transmitter licenses (50 states) | No payment processing without license |
| Commodity Trading (carbon credits, token arbitrage) | CFTC | Carbon credits may be commodity futures |
| Investment Advice (arbitrage, yield analytics) | SEC Investment Advisers Act | RIA registration required |
| Banking Claims | Federal Reserve, OCC, state banking regulators | Cannot claim "bank" status without charter |
| Consumer Protection (NFT experiences, athlete tokens) | FTC, state AGs | Material claims must be substantiated |
| Data Privacy (AI analytics, fan data) | FTC, GDPR, CCPA | User consent and data handling compliance |
| Tax (token gains, NFT transactions) | IRS + state revenue departments | All token transactions may create taxable events |
| Employment / NIL (athlete compensation tokens) | NLRB, state labor law, NCAA regulations | Athlete payments have strict legal framework |
