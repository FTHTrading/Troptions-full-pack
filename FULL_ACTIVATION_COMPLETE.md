```bash
cd troptions-system

# Edit .env with your credentials:
# - TELEGRAM_BOT_TOKEN
# - ISSUER_SEED
# - ISSUER_ADDRESS

# - XRP_WALLET_SEED
```

```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup systemd
```

## рҹ”„ USDC BASE INTEGRATION

**How it works:**
1. User deposits USDC on Base вҶ’ `0xYourBridgeWallet`
2. Relay detects Transfer event
3. Compliance screens transaction
4. Mints USDC-IOU on XRPL 1:1
5. USDC now tradable in all pools
6. Arbitrage bots trade USDC vs TROPTIONS-USD
7. Revenue flows to treasury

