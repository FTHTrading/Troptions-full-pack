  1. Open with IOU honesty: ~874M issued on XRPL/Stellar is demand proof on ledger — not bank reserves or Circle native USDC.
  2. Show PROVEN stack: Academy Stripe, launch.unykorn.org, x402 health, Polygon/KENNY, issuer wallets — GitHub Pages proof pack.
  3. State PIPELINE clearly: fiat-rails stubs on :4022–:4027 respond /health; no production wire until omnibus + BIC + FedWire live.
  4. Ask partner for: omnibus account, FedWire routing, SWIFT BIC or correspondent path, RMA matrix, webhook format, BSA reliance letter scope.
  5. Revenue: issuance/redemption fees (0.25% modeled), float, B2B SWIFT — all PIPELINE until first settled wire hits GL.
